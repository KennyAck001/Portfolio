GitHub Profile Viewer

The GitHub Profile Viewer is a simple and clean web application that allows users to search for any GitHub profile and view key information such as the user's name, avatar, number of public repositories, and followers. It also includes a “View Profile” button that redirects users directly to the person's GitHub profile page for more details.

Notes App

This is a simple and efficient Notes App that lets you write and manage your personal notes directly in the browser. It uses local storage, so your notes are automatically saved and stay even after you refresh or close the tab. You can add multiple notes, and also delete any note whenever you want.

Image Finder

This is an Image Finder App that uses a dummy API to fetch and display image data dynamically. You can search for any keyword, and the app will show relevant images along with their descriptions and photographer details. It's a simple yet functional project that demonstrates API integration and data rendering.

Paytm Clone Using Tailwind CSS

This is a Paytm home page clone built entirely with Tailwind CSS, focusing purely on layout and styling without any JavaScript. The design closely mimics the original, showcasing responsive sections, modern UI elements, and a clean structure. It's a great example of how powerful Tailwind can be for building complex layouts using just utility-first CSS.

Inventory Management

This is a Python-based Inventory Management System that uses an SQLite database to store and manage product data. You can easily add, update, delete, and view inventory items through a simple interface. It's designed to help keep track of stock levels efficiently with persistent local storage.
